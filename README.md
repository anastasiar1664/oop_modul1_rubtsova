Краткое описание приложения "Библиотека"
Класс Book (Книга)

Хранит информацию: название, жанр, год, автор, статус (в наличии/выдана)

Класс Library (Библиотека)

Хранит коллекцию книг

Умеет добавлять книги (метод AddBook)

Что делает программа:

Создаёт 10 книг с разными параметрами

Создаёт библиотеку "Шкаф №1"

Добавляет все книги в библиотеку.
